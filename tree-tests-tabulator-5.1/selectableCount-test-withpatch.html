{ selectable: "highlight" }
This implies an unlimited number of selections. Selection is done by clicking on the checkboxes, but clicking on the rows (except in the checkbox cells) doesn't affect the selection. 
{ selectable: true }
This enables an unlimited number of selections. Clicking on the rows also affects the selection. 
{ selectable: 1 }
This limits the selection count to 1. Clicking on the rows also affects the selection. 
{ selectable: "highlight", selectableCount: 1 }
Selection is done by clicking on the checkboxes, but clicking on the rows (except in the checkbox cells) doesn't affect the selection.
  • without patch: we can't limit the number of selected items
  • with patch: this can be limited (1 in this example)
